# Comprendre Ebola

Ebola awareness game, structured in two chapters: Origin of Ebola and Recognize & Act (symptoms, treatments and prevention). This game is released under the [Creative Commons Attribution 4.0 International (CC BY 4.0) license](https://creativecommons.org/licenses/by/4.0/deed).

The game EGF is embeded using Embedded EGF 1.1 Reader.
